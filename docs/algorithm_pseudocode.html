1. Classical Neural Gas

Online
Fixed number of neurons
Competitive Hebbian graph

Core idea: rank all neurons by distance to the current sample, then move every neuron toward the sample with a rank-dependent exponential neighborhood weight. The winner and runner-up define the graph edge update.

Main State

  • weights[n_neurons, input_dim]
  • adj[n_neurons, n_neurons] with edge ages, -1 meaning no edge
  • step for optional annealing of learning rate and lambda

Main Intuition

  • The winner moves the most.
  • Nearby ranks also move, but less and less.
  • The graph is not learned from gradients; it is updated explicitly from winner pairs.
if init_points is provided:
    initialize weights from init_points
else if input_dim is known:
    initialize weights randomly
initialize adj matrix with -1 everywhere
step = 0

procedure UPDATE(sample):
    sample = coerce sample to shape [D]
    if model not initialized:
        initialize weights using sample dimension

    distances = distance from sample to every neuron
    rank = argsort(distances)            # winner is rank[0]

    (lr_now, lambda_now) = annealed values based on current step

    for position p in 0 .. n_neurons-1:
        neuron_index = rank[p]
        influence = exp(-p / lambda_now)
        weights[neuron_index] += lr_now * influence * (sample - weights[neuron_index])

    if topology updates are enabled:
        age all existing edges by +1
        winner = rank[0]
        runner_up = rank[1]
        set edge(winner, runner_up) age to 0
        delete edges whose age > max_edge_age
        keep diagonal invalid

    step += 1
    return adj, weights

procedure FIT(data, epochs, shuffle):
    repeat for each epoch:
        optionally shuffle samples
        for each sample in data:
            UPDATE(sample)

2. Inverse Neural Gas

Online
Fixed number of neurons
Inverse-rank neighborhood

This is structurally almost the same as classical Neural Gas, but the neighborhood influence is 1 / rank^2 instead of an exponential decay. That creates a sharper winner-focused update.

What Changes vs Neural Gas

  • No annealing state in this implementation.
  • Rank starts effectively at 1, not 0, when computing influence.
  • Topology update logic is otherwise the same.

Mental Model

  • The winner still dominates.
  • Second and third best neurons still move, but much less.
  • Far-away ranks quickly become almost irrelevant.
if init_points is provided:
    initialize weights from init_points
else if input_dim is known:
    initialize weights randomly
initialize adj matrix with -1 everywhere

procedure UPDATE(sample):
    sample = coerce sample to shape [D]
    if model not initialized:
        initialize weights using sample dimension

    distances = distance from sample to every neuron
    rank = argsort(distances)

    for position p in 0 .. n_neurons-1:
        neuron_index = rank[p]
        inverse_rank = p + 1
        influence = 1 / (inverse_rank^2)
        weights[neuron_index] += lr * influence * (sample - weights[neuron_index])

    if topology updates are enabled:
        age all existing edges by +1
        winner = rank[0]
        runner_up = rank[1]
        set edge(winner, runner_up) age to 0
        delete edges whose age > max_edge_age
        keep diagonal invalid

    return adj, weights

3. Growing Neural Gas

Online
Adaptive number of neurons
Error-driven node insertion

This model starts from two nodes and evolves both the graph and the number of nodes. It keeps a local error value per node, uses winner and neighbor learning rates, removes stale edges, removes isolated nodes, and periodically inserts a new node in a high-error region.

Main State

  • weights[n_nodes, D]
  • errors[n_nodes]
  • adj[n_nodes, n_nodes] with edge ages
  • steps to decide when to grow

Main Intuition

  • Move the winner strongly, neighbors weakly.
  • Track where approximation error accumulates.
  • Insert new nodes where the current graph struggles most.
procedure INITIALIZE(input_dim, optional seed_sample, optional init_points):
    if init_points exists:
        create one active node per provided point
        initialize a sparse starting graph among those nodes
    else if seed_sample exists:
        center = seed_sample
    else:
        center = random point

    create 2 initial nodes near center using small random jitter
    errors = zeros(2)
    adj = 2x2 matrix with edge(0, 1) = edge(1, 0) = 0

procedure UPDATE(sample):
    sample = coerce sample to shape [D]
    if model not initialized:
        INITIALIZE(sample dimension, seed_sample = sample)

    distances = distance from sample to every node
    s1 = winner index
    s2 = second-winner index

    age all edges connected to s1 by +1
    connect s1 and s2 with age 0

    error_term = distance to s1
    if using l2 distance:
        error_term = error_term^2
    errors[s1] += error_term

    weights[s1] += lr_winner * (sample - weights[s1])
    for each neighbor n of s1:
        weights[n] += lr_neighbor * (sample - weights[n])

    remove edges older than max_edge_age
    remove isolated nodes, but keep at least 2 nodes alive

    steps += 1
    if steps mod lambda_steps == 0:
        INSERT_NEW_NODE()

    errors *= (1 - beta)
    return adj, weights

procedure INSERT_NEW_NODE():
    if n_nodes already equals max_neurons:
        stop

    q = node with largest accumulated error
    neighbors = neighbors of q
    if q has no neighbors:
        stop

    f = neighbor of q with largest error
    r_weight = 0.5 * (weights[q] + weights[f])
    append new node r with weight r_weight

    new_error = 0.5 * (errors[q] + errors[f])
    append new node error = new_error

    remove edge(q, f)
    connect q to r
    connect f to r

    errors[q] *= alpha
    errors[f] *= alpha
    errors[r] = new_error

4. Differentiable Neural Gas

Differentiable
Optimized with backpropagation
Soft rank approximation

This is not an online winner-take-most update. Instead, the neuron weights are trainable parameters inside an nn.Module. The model computes a fully differentiable loss using soft ranks obtained from pairwise sigmoid comparisons.

Main State

  • weights as nn.Parameter
  • No discrete adjacency matrix
  • No explicit update(sample) method

Main Intuition

  • Approximate rank smoothly so gradients can flow.
  • Turn rank into a neighborhood weighting.
  • Minimize a weighted distance loss with standard optimizers.
weights = learnable parameter matrix [n_neurons, D]

procedure FORWARD(batch):
    x = coerce batch to shape [N, D]
    distances = pairwise distance between x and weights   # [N, n_neurons]

    soft_rank = SOFT_RANK(distances)
    neighborhood = NEIGHBORHOOD_WEIGHT(soft_rank)

    loss = mean(neighborhood * distances)
    return loss

procedure SOFT_RANK(distances):
    for each sample and for each neuron pair (i, j):
        prob_i_greater_than_j = sigmoid((distance_i - distance_j) / temperature)

    ignore i == j comparisons
    soft_rank_i = 1 + sum_j prob_i_greater_than_j
    return soft_rank

procedure NEIGHBORHOOD_WEIGHT(soft_rank):
    if neighborhood == exponential:
        return exp(-(soft_rank - 1) / lambda_value)
    else if neighborhood == inverse:
        return 1 / soft_rank^2

training loop outside the module:
    optimizer.zero_grad()
    loss = model(batch)
    loss.backward()
    optimizer.step()

5. Differentiable Growing Neural Gas

Differentiable core
Discrete growth step
Learnable soft topology

This model blends two ideas: differentiable prototype learning and an explicit GNG-like growth operation. The forward pass learns both prototypes and a soft edge matrix, while grow() is a separate discrete structural step.

Main State

  • weights[max_neurons, D] as learnable parameters
  • edge_logits[max_neurons, max_neurons] as learnable parameters
  • active_mask deciding which nodes are currently alive
  • node_error_ema to decide where to grow next

Main Intuition

  • Use soft ranks like differentiable NG.
  • Use learnable edge probabilities to inject topology into the neighborhood weighting.
  • Use a separate growth rule driven by an EMA of node difficulty.
weights = learnable parameter matrix [max_neurons, D]
edge_logits = learnable parameter matrix [max_neurons, max_neurons]
if init_points is provided:
    first K rows of weights come from init_points
    active_mask marks those K nodes active
else:
    active_mask = first init_neurons are active
node_error_ema = zeros(max_neurons)

procedure EDGE_PROB_MATRIX():
    logits_sym = 0.5 * (edge_logits + transpose(edge_logits))
    probs = sigmoid(logits_sym)
    zero out diagonal
    return probs

procedure FORWARD(batch):
    x = coerce batch to [N, D]
    idx = active node indices
    w = weights[idx]

    distances = pairwise distance between x and active weights
    soft_rank = SOFT_RANK(distances)
    neighborhood = NEIGHBORHOOD_WEIGHT(soft_rank)

    edge_prob = EDGE_PROB_MATRIX restricted to active nodes

    if there is more than one active node:
        degree = row-sum(edge_prob) / (n_active - 1)
    else:
        degree = 0

    neighborhood = neighborhood * (1 + topology_influence * degree)
    data_term = mean(neighborhood * distances)

    proto_dist = squared l2 distances among active prototypes
    weighted_len = sum(edge_prob * proto_dist) / (sum(edge_prob) + small_constant)
    sparsity = mean(edge_prob)

    loss = data_term
         + edge_length_coeff * weighted_len
         + edge_sparsity_coeff * sparsity

    with no gradient:
        node_err = mean over batch of neighborhood * distances
        node_error_ema[active] =
            (1 - beta) * node_error_ema[active] + beta * node_err

    return loss

procedure GROW(n_new = 1):
    repeat up to n_new times:
        if active nodes already reach max_neurons:
            stop

        q = active node with largest node_error_ema

        choose f:
            first try the active node most strongly connected to q
            if no positive soft edge exists, choose nearest active prototype instead

        choose first inactive slot r
        set weights[r] = midpoint(weights[q], weights[f]) + optional noise
        mark r active

        weaken edge(q, f) strongly
        initialize edges (q, r) and (f, r) with edge_init_logit

        new_err = 0.5 * (node_error_ema[q] + node_error_ema[f])
        node_error_ema[q] *= grow_error_decay
        node_error_ema[f] *= grow_error_decay
        node_error_ema[r] = new_err

        count one new node added

Big Picture Comparison

Online Family

  • NeuralGas: fixed-size, exponential neighborhood.
  • InverseNeuralGas: fixed-size, inverse-squared-rank neighborhood.
  • GrowingNeuralGas: variable-size, local error drives growth.

Differentiable Family

  • DifferentiableNeuralGas: replaces hard rank with soft rank and uses SGD-style training.
  • DifferentiableGrowingNeuralGas: adds learnable soft topology and a discrete grow() step.
